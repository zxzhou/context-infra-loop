# Skills Index

This index lists reusable skills in the context infrastructure system.

- [Multi-Agent Analysis Best Practices](bestpractice_multi_agent_analysis.md): Split analytical work only when scopes are independent, responsibilities are clear, and integration has an explicit synthesis step.

- [Temporal Information Verification](bestpractice_temporal_info_verification.md): For current or date-sensitive information, verify source freshness and state concrete dates rather than relying on memory.
- [Delayed Execution](delayed_execution.md): Schedule future work with clear trigger time, context, expected action, and completion criteria.
- [Google Docs Workflow](google_docs.md): Work with Google Docs-targeted artifacts through structured editing, comments, review, and export-safe formatting.
- [Growth Analytics](growth_analytics.md): Analyze acquisition, activation, retention, revenue, and referral with cohort thinking and actionable diagnostics.

- [Semantic Search](semantic_search.md): Index Markdown knowledge, embed chunks, and retrieve relevant context by meaning while preserving source references.
- [Send Email](send_email.md): Send concise, well-formatted email through the configured local mail utility with clear subject and body.
