# Observations

This file is the L1/L2 memory layer for the context infrastructure system. Daily observations are appended by `periodic_jobs/ai_heartbeat/src/v0/observer.py`; periodic reflection is handled by `reflector.py`.

## Format

Each entry uses this shape:

```markdown
## Date: YYYY-MM-DD

- High: reusable lesson, hard constraint, or architecture-level decision.
- Medium: active project context or decision background likely to matter soon.
- Low: transient task notes, debug traces, or temporary context.
```

## Loading Guidance

Do not load the whole file unless needed. Search by date, project, or topic, or use semantic search when the exact wording is unknown.
