# Skill Ecosystem

Skills are reusable operating procedures. They sit between high-level axioms and one-off task notes. A good skill tells an agent when to use it, what steps to follow, what quality bar to meet, and what pitfalls to avoid.

## Organization

- `rules/skills/INDEX.md` lists available skills.
- `rules/skills/bestpractice_*` files capture reusable best practices.
- `rules/skills/workflow_*` files capture repeatable multi-step workflows.

## Maintenance

Promote a workflow into a skill only after it has repeated value. Keep skills concise, operational, and testable. Retire or merge skills that overlap heavily.
